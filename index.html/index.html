1.1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

La etiqueta UL sirve para crear una lista no ordenada. Se tienen que usar si o si con la etiqueta "li". Lo utilizaría para hacer una lista de compras.

  • Huevos
  • Fideos
  • Harina

1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

La etiqueta OL sivrve para crear una lista ordenada, ya sea de manera numerica o alfabetica. Esta tambien se debe usar si o si con la etiqueta "li" Lo utilizaría para responder preguntas.

  1. pregunta uno
  2. pregunta dos
  3. pregunta tres

1.3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

DL sirve para representar una lista descriptiva, encierra terminos y descripciones las cuales se las dará las etiquetas "dt" y "dd", de uso obligatorio para la etiqueta "dl". La ultilizaría para dar diferentes conceptos o datos de un mismo tema.

nombre de la persona

1.4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

LI sirve para crear elementos de una lista. Siempre se debe utilizar con "ul" u "ol". Lo usaría para hacer una lista de materias.

  • Lengua
  • Ingles
  • Lógica
